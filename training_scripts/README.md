# Training Scripts

`training_scripts/` es una capa historica o semioperativa de runners versionados.

Reglas de lectura:

- no es la API canonica del repo;
- las ultimas `1-2` versiones por familia suelen ser la mejor referencia operativa;
- el resto debe tratarse como historico salvo evidencia contraria;
- antes de mostrar o reutilizar un script, revisar dependencias locales, rutas y supuestos metodologicos.

Esta carpeta se conserva por trazabilidad y rescate de logica, no como superficie publica principal.
